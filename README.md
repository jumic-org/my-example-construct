# Welcome to My Example Construct

This example project shows how to develop and publish a construct. It uses:

- `release-please` GitHub for semantic versioning
- AWS CodeArtifact connected with OpenID Connect as artifact repository
- Automated Dependabot updates
- An GitHub app for automation (without personal access token)
- GitHub branch protection with auto-merge enabled
- ESLint and Prettier for linting and formatting
